# AI Prompter - macOS Application

## Installation

### Option 1: Drag & Drop (Easiest)
1. Open **Finder**
2. Locate this folder
3. Drag `AI Prompter.app` to your **Applications** folder
4. Done! Now you can launch it from Launchpad or Applications

### Option 2: Using Script
```bash
cd dist-app
./install.sh
```

### Option 3: Command Line
```bash
open dist-app/AI\ Prompter.app
```

## Running the App

Once installed, you can:
- **Launch from Applications folder** - Just like any other Mac app
- **Launch from Spotlight** - Press Cmd+Space, type "AI Prompter"
- **Launch from Terminal** - Run: `open /Applications/AI\ Prompter.app`

## First Time Running

The app will:
1. Start Next.js server automatically (localhost:3000)
2. Open app window
3. Load the web interface

Data is stored locally in `~/Library/Application Support/AI Prompter/`

## Uninstall

Simply drag `AI Prompter.app` from Applications to Trash.
